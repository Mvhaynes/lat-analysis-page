Latitude Analysis Dashboard

This dashboard uses latitudes obtained from city data and plots the coordinate data against each city's maximum temperature, humidity, cloudiness, and wind speed. The purpose of this analysis was to determine how weather might be affected by distance from the equator.

This analysis was created using randomly generated coordinate data passed through the CitiPy module to obtain city names and the OpenWeatherMap API to obtain city weather data. Weather data was plotted using Matplotlib. You can see the full code on GitHub.
